## About The Project

This is a responsive personal portfolio website built to present my profile as a Full Stack Developer. The website highlights my technical skills, internship experience, selected projects, certifications, and professional contact details in a clean and modern one-page layout.

The design is optimized for both desktop and mobile screens, with smooth interactions, polished visual styling, and a professional presentation suitable for recruiters, clients, and internship opportunities.

## Features

- Responsive design for laptop, tablet, and mobile
- Premium one-page portfolio layout
- Modern hero section with strong visual hierarchy
- Skills, experience, projects, and credentials sections
- Resume download support
- Internship project live website link
- Smooth scroll navigation and active section highlighting
- Scroll progress indicator and subtle motion effects
- SEO-friendly meta tags, Open Graph tags, and structured data

## Getting Started

To run this project locally:

1. Clone the repository:

```bash
https://github.com/darshandobariya563/portfolio.git
```

2. Open the project folder.

3. Run a local server, for example:

```bash
python -m http.server 8000
```

4. Open in your browser:

```text
http://127.0.0.1:8000
```
